# LUTLINE

Parse `argv` using a look-up table generated from your command-line interface specification.

## Install

Install lutline from PyPI with:

```
$ pip install lutline
```

## Development

### Test
Test lutline using:

```
$ python -m tests
```

### Deployment

Deploy to PyPI with:

```
$ python setup.py sdist bdist_wheel
$ twine upload dist/*
```
